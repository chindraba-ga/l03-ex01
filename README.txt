Lesson 3: Partner Exercise: Practicing the Box Model

    1.  Draw a DOM tree to help lay this page out:

        https://s3-us-west-2.amazonaws.com/s.cdpn.io/2522641/wireframe-basic.pdf

    2.  Apply the box model CSS elements to the HTML in the sample. Adjust the
        box model to test what happens.

    3.  If you're feeling really ambitious, try to align the three elements
        horizontally.
